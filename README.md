# tcdw

`tcdw` 的所有含义。

## 在 JavaScript 项目中使用

```javascript
const tcdw = require('tcdw');

// print a random meaning
console.log(`tcdw means ${tcdw[Math.floor(Math.random() * tcdw.length)]}`);
```

## 贡献

1. 编辑 `index.txt`，增加新的含义。一个含义一行（UNIX 换行符）。不要直接编辑 `index.js`！
2. 运行 `make_json.js`
3. 执行 git 相关操作
4. Pull Request
